Introduction à HTML

L'HTML a été créé par Tim Berners-Lee, un ingénieur britannique, en 1991. Il a développé cette technologie pour permettre aux chercheurs du monde entier de partager facilement des documents et des informations via Internet.

Tim Berners-Lee a conçu l'HTML dans le but de structurer et de présenter des informations sur le web. Il voulait créer un langage qui soit simple à utiliser, permettant aux utilisateurs de créer des pages web sans avoir besoin de connaissances techniques approfondies. L'objectif principal était de faciliter ainsi la navigation, la mise en page, et le partage d'informations.

HTML (HyperText Markup Language) est le langage utilisé pour créer des pages web. Il permet de structurer du contenu en utilisant des balises et d'ajouter des attributs pour enrichir ces éléments.

Les balises HTML

Les balises HTML sont des éléments qui définissent la structure et le contenu d'une page web. Voici quelques-unes des balises les plus courantes :

  • <h1> à <h6> : balises de titre
  • <p> : balise de paragraphe
  • <a> : balise de lien
  • <img> : balise d'image
  • <div> : balise de division

Attributs HTML

Les attributs HTML sont des informations supplémentaires que l'on peut ajouter aux balises pour en modifier le comportement ou l'apparence. Voici quelques attributs courants :

  • class : permet de définir une classe CSS pour un élément
  • id : permet de définir un identifiant unique pour un élément
  • src : spécifie la source d'une image ou d'un fichier multimédia
  • href : spécifie l'URL d'un lien

Exemple de code HTML

            
                <!DOCTYPE html>
                <html lang="fr">
                <head>
                    <meta charset="UTF-8">
                    <title>Exemple de page HTML</title>
                </head>
                <body>
                    <h1>Bienvenue sur ma page web</h1>
                    <p>Ceci est un paragraphe d'exemple.</p>
                    <a href="https://www.example.com">Visitez mon site</a>
                </body>
                </html>
